# Spoticool

## Description

Spoticool est un site qui présente les musiques et artistes préférés de Charly et Emy.

## Prerequisites

Pour lancer Spoticool sans erreur, il faut impérativement le lancer sur un serveur local (Vous pouvez utiliser l'extention Live Server de Visual Studio Code)

## Installation

1. Clonez le projet sur votre ordinateur
2. Ouvrez le projet dans Visual Studio Code
3. Faite un clique droit sur le fichier index.html et cliquez sur "Open with Live Server". Le site devrait s'ouvrir dans votre navigateur par défaut.
